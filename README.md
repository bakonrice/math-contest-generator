# Math Contest Problem Generator

A Flask-based web application that generates contest-style math problems
and evaluates user responses.

The backend is implemented in Python using Flask, while the frontend
uses HTML and CSS only.

## Current Features
- Randomly generated linear equation problems
- Server-side answer validation

## Planned Features
- Quadratic and modular arithmetic problems
- Difficulty levels
- Score tracking

## Tech Stack
- Python (Flask)
- HTML / CSS
